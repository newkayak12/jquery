속성 관리하기

attr()함수

                    attr(): 태그가 가지고 있는 속성에 대해서 수정/확인 할 수 있는 함수이다.

                    1.attr("속성명") : 속성명의 값을 확인한다. 

                    2.attr("속성명", "속성값") : 속성명에 속성값을 대입한다.

                    3.attr("속성명", function( (i,v) =>{ ... } )) : 속성명에 함수 로직을 실행해서 반환하는 속성 값을 대입한다. 

                    4.attr({속성명: 속성값, 속성명: 속성값, ... }) : 객체에 지정된 속성, 속성값을 대입

                    5.removeAttr("속성명") : 속성명에 해당하는 속성을 삭제
